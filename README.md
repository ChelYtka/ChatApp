# ChatApp — Сетевое чат-приложение

## О проекте

**ChatApp** — десктопное чат-приложение для локального общения между пользователями на одном компьютере. Разработано на C++ и Qt 6 в рамках курсовой работы по дисциплине «Технологии программирования» (ТПУ, группа 8В32).

Приложение позволяет нескольким пользователям вести переписку, создавать чаты, обмениваться сообщениями и управлять профилями — всё в пределах одного ПК.

---

## Основные функции

- **Управление профилями** — создание, удаление, переключение между пользователями.
- **Личные чаты** — создание чатов между двумя пользователями, автоматическое переключение на существующий чат.
- **Отправка сообщений** — через кнопку «Отправить» или клавишу Enter.
- **История сообщений** — все сообщения сохраняются в JSON-файл и загружаются при следующем запуске.
- **Удаление сообщений** — выберите сообщение и нажмите «Удалить» (с подтверждением).
- **Удаление чата** — полностью очищает историю переписки.
- **Визуальное различие** — сообщения текущего пользователя выделены зелёным фоном (справа), собеседника — белым (слева). Отображаются имя и время отправки.

## Сборка и запуск

### Linux (Ubuntu/Debian)

```bash
# Установить Qt
sudo apt update
sudo apt install qt6-base-dev build-essential

# Собрать и запустить
cd ChatApp
qmake ChatApp.pro
make
./ChatApp
```

---

### Windows

**1. Установить Qt 6** с сайта [qt.io](https://www.qt.io/download) (выбрать компонент Qt Multimedia).

**2. Запустить через Qt Creator:**

- Открыть `ChatApp.pro` → Configure Project → ▶ Run.

**3. Или через командную строку (MinGW):**

```cmd
cd ChatApp
qmake ChatApp.pro
mingw32-make
ChatApp.exe
```
